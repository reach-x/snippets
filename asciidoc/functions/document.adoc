== Introduction

This document demonstrates *AsciiDoc* markup features. AsciiDoc is a
lightweight markup language for authoring notes, articles, documentation,
books, and more. It's more feature-rich than Markdown and easier to write
than DocBook.

== Text Formatting

=== Basic Formatting

- *Bold text* (strong emphasis)
- _Italic text_ (emphasis)
- `Monospace text` (code)
- *_Bold and italic_*
- `*Bold monospace*`

=== Advanced Formatting

- ^Superscript^: E = mc^2^
- ~Subscript~: H~2~O
- [.line-through]#Strikethrough#
- [.underline]#Underline#
- #Highlighted text#

=== Source Code Listings

[source,python]

def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

print(factorial(5))
----
